How To Use Oppeneigen Image Explorer

OpenEigen Image Explorer is a Content-Based Image Retrieval (CBIR) system designed to search images based on their color and texture features. Operating with an existing dataset, the system showcases images matching selected features with a similarity threshold of over 60%. Efficiently locate visually similar images with ease!

1. Upload Ur Dataset
- Folder of Image - Scrape Image from URL Site Waiting for Database Existence
2. Upload Query Image 3. Choose CBIR feature
- Color (recommended) - Texture
3. Click Search button 4. Download the report result
